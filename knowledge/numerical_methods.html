1. ニュートン法 (Newton-Raphson Method)

ニュートン法は、非線形方程式 f(x) = 0 の根を反復的に求めるアルゴリズムです。 初期値 x₀ から出発し、以下の漸化式で解を更新していきます。

x_{n+1} = x_n - f(x_n) / f'(x_n)

収束速度は2次収束で、初期値が十分に解に近ければ非常に高速に収束します。 ただし、f'(x) = 0 となる点では適用できないため注意が必要です。

Python 実装例

def newton_method(f, df, x0, tol=1e-10, max_iter=100): """ニュートン法で f(x) = 0 の根を求める""" x = x0 for i in range(max_iter): fx = f(x) dfx = df(x) if abs(dfx) < 1e-15: raise ValueError("導関数が0に近すぎます") x_new = x - fx / dfx if abs(x_new - x) < tol: return x_new, i + 1 # 解と反復回数 x = x_new raise ValueError("収束しませんでした") # 例: x^2 - 2 = 0 の根 (√2 を求める) f = lambda x: x**2 - 2 df = lambda x: 2*x root, iters = newton_method(f, df, x0=1.0) print(f"√2 ≈ {root:.15f} ({iters}回の反復)")

2. 連立一次方程式の解法

連立一次方程式 Ax = b の解法にはいくつかの手法があります。

  • ガウスの消去法: 前進消去と後退代入による直接法
  • LU分解: A = LU に分解して効率的に解く
  • ヤコビ法: 対角成分を使った反復法
  • ガウス-ザイデル法: ヤコビ法の改良版（更新値を即座に利用）

ガウスの消去法 (Python)

import numpy as np def gauss_elimination(A, b): """ガウスの消去法で Ax = b を解く""" n = len(b) # 拡大係数行列を作成 Ab = np.hstack([A.astype(float), b.reshape(-1, 1).astype(float)]) # 前進消去 for i in range(n): # ピボット選択（部分ピボッティング） max_row = np.argmax(np.abs(Ab[i:, i])) + i Ab[[i, max_row]] = Ab[[max_row, i]] for j in range(i + 1, n): factor = Ab[j, i] / Ab[i, i] Ab[j, i:] -= factor * Ab[i, i:] # 後退代入 x = np.zeros(n) for i in range(n - 1, -1, -1): x[i] = (Ab[i, -1] - np.dot(Ab[i, i+1:n], x[i+1:])) / Ab[i, i] return x

3. 補間法

与えられたデータ点を通る多項式を構築する手法です。

  • ラグランジュ補間: 基底多項式の線形結合
  • ニュートン補間: 差分商を利用した逐次的な構築
  • スプライン補間: 区分的多項式による滑らかな補間

ラグランジュ補間

def lagrange_interpolation(x_data, y_data, x): """ラグランジュ補間""" n = len(x_data) result = 0.0 for i in range(n): term = y_data[i] for j in range(n): if i != j: term *= (x - x_data[j]) / (x_data[i] - x_data[j]) result += term return result

4. 数値積分

解析的に積分できない関数に対して、近似的に定積分を計算する方法です。

  • 台形公式: 区間を台形で近似。精度は O(h²)
  • シンプソン則: 放物線で近似。精度は O(h⁴)
  • ガウス求積法: 最適な点と重みを使う高精度な手法

シンプソン則 (Python)

def simpson_rule(f, a, b, n=100): """シンプソン則で ∫[a,b] f(x) dx を近似""" if n % 2 == 1: n += 1 # n は偶数にする h = (b - a) / n s = f(a) + f(b) for i in range(1, n): x = a + i * h s += 4 * f(x) if i % 2 == 1 else 2 * f(x) return s * h / 3 # 例: ∫[0, π] sin(x) dx = 2 import math result = simpson_rule(math.sin, 0, math.pi, n=100) print(f"結果: {result:.15f}")

5. 常微分方程式の数値解法

常微分方程式の初期値問題 y' = f(t, y), y(t₀) = y₀ を数値的に解く方法です。

  • オイラー法: 1次精度の最も基本的な手法
  • ルンゲ-クッタ法 (RK4): 4次精度の標準的手法

4次ルンゲ-クッタ法 (Python)

def rk4(f, t0, y0, t_end, dt): """4次ルンゲ-クッタ法""" t, y = t0, y0 results = [(t, y)] while t < t_end: k1 = f(t, y) k2 = f(t + dt/2, y + dt/2 * k1) k3 = f(t + dt/2, y + dt/2 * k2) k4 = f(t + dt, y + dt * k3) y = y + (dt / 6) * (k1 + 2*k2 + 2*k3 + k4) t += dt results.append((t, y)) return results
